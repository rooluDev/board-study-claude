## 🎯 프로젝트 소개

이 프로젝트는 **Servlet/JSP를 활용한 전통적인 게시판 시스템**으로, Spring Framework 없이 순수 Java 웹 기술로 구현되었습니다.

### 프로젝트 목적

- Servlet/JSP 기본 아키텍처 학습
- MyBatis를 활용한 데이터베이스 연동
- 파일 업로드/다운로드 구현
- **AI(Claude)와의 협업 개발 방법론 실험**

### 개발 방법론의 특징

본 프로젝트는 **반복적 개선(Iterative Refinement)** 방식으로 개발되었습니다:

1. **claude1 브랜치**: 초기 구현 및 기본 기능 개발
2. **claude2 브랜치**: 1차 리팩토링 및 문서 개선
3. **claude3 브랜치**: 2차 리팩토링 및 최종 완성
4. **main 브랜치**: 프로젝트 전체 가이드 및 통합 문서

각 브랜치마다 PRD(요구사항 문서)와 프롬프트를 발전시키며, AI와의 협업 효율을 높였습니다.
