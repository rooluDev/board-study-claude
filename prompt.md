# 게시판 프로젝트 생성 프롬프트

## 초기 프롬프트

```
Servlet/JSP 기반의 게시판 시스템을 구현하고자 합니다. 

프로젝트 구조:
- Servlet, Service, DAO, MyBatis Mapper 계층 분리
- Java 17, Tomcat 10.1.50, MySQL 8.0.31 사용
- Gradle 빌드 도구 사용
- Spring Framework 사용 불가

주요 기능:
1. 게시글 목록 조회 (페이징, 검색)
2. 게시글 상세 조회 (조회수 증가)
3. 게시글 등록 (첨부파일 최대 3개)
4. 게시글 수정 (비밀번호 인증)
5. 게시글 삭제 (비밀번호 인증)
6. 댓글 등록
7. 첨부파일 다운로드

프로젝트 루트에 있는 prd.md 파일에 전체 요구사항이 상세히 명시되어 있습니다.
CLAUDE.md 파일에는 프로젝트 진행 가이드가 있습니다.
convention.md 파일에는 코딩 컨벤션이 정의되어 있습니다.

위 문서들을 먼저 읽고, 프로젝트 구조를 제안한 후 단계별로 구현해주세요.
첫 단계로 Gradle 프로젝트 초기 설정과 디렉토리 구조를 생성해주세요.
```

## 진행 중 사용할 프롬프트 예시

### 1단계 완료 후
```
프로젝트 구조가 잘 생성되었습니다. 
다음 단계로 데이터베이스 설정을 진행해주세요:
1. MyBatis 설정 파일 (mybatis-config.xml)
2. 데이터베이스 연결 설정
3. DDL 스크립트 작성 (테이블 생성)
```

### 기능 구현 요청 시
```
게시글 목록 조회 기능을 구현해주세요.
prd.md의 "3.1 게시글 목록 조회" 섹션을 참고하여:
- BoardListServlet (GET /boards)
- BoardService의 목록 조회 메서드
- BoardDAO와 BoardMapper.xml
- list.jsp 페이지
를 구현해주세요.
```

### 오류 해결 요청 시
```
[오류 메시지/현상 설명]

prd.md의 요구사항과 convention.md의 컨벤션을 준수하여 
문제를 해결해주세요.
```

## 주의사항

1. 한 번에 너무 많은 기능을 요청하지 말 것
2. 각 단계별로 결과를 확인하고 다음 단계로 진행
3. prd.md의 요구사항을 정확히 따르도록 명시
4. convention.md의 코딩 스타일 준수 요청
5. 검증 및 예외 처리를 빠뜨리지 않도록 주의
